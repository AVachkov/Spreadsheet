# Spreadsheet (Електронни таблици)

Конзолно приложение за работа с електронни таблици, разработено на C++ като курсов проект по Обектно-ориентирано програмиране (ФМИ, СУ).

## Поддържана функционалност
* Четене и запис на таблици от текстови файлове.
* Поддръжка на типове: цели числа, дробни числа, текст и формули.
* Оценяване на математически формули (събиране, изваждане, умножение, деление, степенуване).
* Рефериране на клетки (напр. `=R1C1 + R2C2`).
* Засичане на циклични зависимости (circular dependency) и невалидни данни.

## Как да компилирате проекта

### Вариант 1: Чрез терминал (Препоръчително)

Отворете терминала в главната директория на проекта и изпълнете:

**Mac / Linux:**
\`\`\`bash
clang++ -std=c++17 -Iinclude src/*.cpp -o build/main
./build/main
\`\`\`

**Windows:**
\`\`\`cmd
g++ -std=c++17 -Iinclude src\*.cpp -o build\main.exe
build\main.exe
\`\`\`

### Вариант 2: Чрез Visual Studio Code
Проектът съдържа конфигуриран `.vscode/tasks.json`. Просто натиснете \`Cmd + Shift + B\` (Mac) или \`Ctrl + Shift + B\` (Windows/Linux), за да компилирате автоматично.

## Как да стартирате
След успешно компилиране, стартирайте програмата с:
\`\`\`bash
./build/main
\`\`\`

## Тестове
В папката `tests/` ще намерите примерни файлове с валидни и невалидни данни (`test_basic.txt`, `test_formulas.txt`, ...), с които можете да тествате командите `open`, `print`, `edit` и `save`.
